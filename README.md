
## Project title
# WOW Media ( Social Media App )

## Motivation
It is an example of how React, Tanstack Query, Appwrite and Tailwind CSS works.
<br>
App enables to user:
- to CRUD operations on posts
- add description, image, location and tags
- like save post
- infinite scroll
- searched saved posts
